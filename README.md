# McGill Course Explorer

Welcome to the McGill Course Explorer project! This website allows users to search for courses offered at McGill University, view detailed information about each course, and manage their course schedule.

## Overview

The McGill Course Explorer provides an intuitive interface for students to find and explore courses. Users can search for courses by code, view official descriptions, read student reviews, and add courses to their personal schedules.

## Project Structure

- `index.html`: Contains the HTML structure for the McGill Course Explorer website, including layout, styles, and scripts for functionality.
- `README.md`: Documentation for the project, including setup instructions and usage details.

## Setup Instructions

1. **Clone the Repository**: 
   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory**:
   ```bash
   cd mcgill-course-explorer-site
   ```

3. **Open the `index.html` File**: You can open the `index.html` file in your web browser to view the website.

## Usage

- Enter a course code in the search bar (e.g., "COMP 202") and click the "Search" button.
- Review the course details, including the official description, ratings, and prerequisites.
- Select a term to register for the course and add it to your schedule.
- Access external links for more information about the course.
